modern tetris with peer2peer pvp (W.I.P)

(note: you will have to run an http server for the website to work properly)

gamemodes: solo freeplay, multiplayer 1v1

in order to connect to a user to 1v1, the process goes as such:

User 1 creates an offer.
User 2 copy pastes the offer into the first input box and creates an answer.
User 1 copy pastes the answer into the second input box and clicks add answer.
